## 📌 Overview
**3D LoadAI** is an open-deployment but proprietarily secured system that connects multiple external AI intelligence sources, merges their understanding, and converts your natural language prompts directly into high-quality 3D models and animations.

This system **does not expose core logic or intellectual property** — external AIs are only used as supplementary insight providers. All generation, security, and ownership remain fully under your control.

---

## ✨ Key Features
- 🧠 **Multi-AI Insight Fusion**: Connect Google Gemini, OpenAI GPT, Anthropic Claude — select which AI to use for each task
- 💬 **Chat-Driven Workflow**: Describe your scene or idea naturally — no complex technical prompts required
- 🎨 **Full 3D Output**: Generates models, textures, rigs, and motion — export as `.glb`, `.fbx`, `.gltf`, or rendered video
- 🔐 **Protected Access**: Locked with master key `3DLoadAI` — cannot be unlocked or modified without authorization
- 🔑 **Your Own API Keys**: You provide your external API keys — no shared keys, no hidden costs
- 📡 **Ready-to-Use API**: Integrate into your own projects, apps, or workflows easily
- 🖥️ **Simple Web Interface**: Set keys, select AIs, and generate directly from your browser

## 🚀 Installation Guide

### 1. Clone or Download
```bash
git clone https://github.com/[YOUR-USERNAME]/3D-LoadAI.git
cd 3D-LoadAI
2. Install Dependencies
 
 pip install -r requirements.txt

 
3. Configure Environment
 
Copy the template file and create your secure configuration:

 cp .env.example .env

 
 
Edit  .env  and add:
 
- Your encrypted master lock value
​
- Your own external API keys (Gemini / GPT / Claude)
 
⚠️ Never commit  .env  to GitHub — it is already excluded in  .gitignore 
 
4. Run the System

 python app.py

 
 
Access the interface at:  http://localhost:8080
